# Living Signal - Inside Cover Cheat Sheet

Author: Aaron Sahlstrom  
Version: 0.0.2

## Entry Format

```
DATE:
LOCATION:
CONTEXT: <one word>

Then write normally.
```

Add a Signal only if something matters:

```
-
Signal: <Type>
<your words>
-
```

That is the entire structure.

## Canonical Signal Types

- Reflection - meaning, insight, perspective shift
- Emotion - felt experience, mood
- Decision - choice, commitment, intent
- Question - open inquiry, uncertainty
- Memory - past resurfacing with meaning
- Identity - who I am / who I am becoming

## Rules

- One context per entry
- Signals are optional and intentional
- Write like a human

## Context Examples

- Daily (default)
- Travel
- Work
- Relationship
- Health
- Insight
